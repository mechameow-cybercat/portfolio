VARIABLES

These are containers that store values, such as numbers, strings, booleans, arrays, objects, etc.

Variables can be declared with keywords like var, let, or const, depending on the scope and mutability of the value.

let a = function () {}; var b = []; const c = {};
let x = 0, y = "string", z = true;
  • Variable in JavaScript are loosely typed.
OPERATORS

These are symbols that perform arithmetic, logical, or assignment operations on values, such as +, -, *, /, %, &&, ||, =, +=, etc.

Operators can be used to manipulate variables and expressions in JavaScript.

let a = 0, b = 1; a == b //false
  • There are also bitwise operators.
CONTROL STRUCTURES

These are statements that control the flow of execution in a program, such as if-else, switch-case, for, while, do-while, break, continue, etc.

Control structures can be used to implement conditional logic and loops in JavaScript.

// If statement to compare the value type of variable x

if (typeof x === "number") {
// If x is a number, display "x is a number" in the console
console.log("x is a number"); }

else if (typeof x === "string") {
// If x is a string, display "x is a string" in the console
console.log("x is a string"); }

else if (typeof x === "boolean") {
// If x is a boolean, display "x is a boolean" in the console
console.log("x is a boolean"); }

else if (Array.isArray(x)) {
// If x is an array, display "x is an array" in the console
console.log("x is an array"); }

else {
// If x is none of the above, display "x is an object" in the console
console.log("x is an object"); }
  • They have negligible performance differences.
FUNCTIONS

These are blocks of code that can be defined and invoked with a name or anonymously. Functions can accept parameters and return values.

Functions can also be assigned to variables or passed as arguments to other functions. Functions are the main way of modularizing and reusing code in JavaScript.

// Create a new image element with the link as the source
// This function takes a hyperlink parameter and displays an image as a popup from that link

function showImagePopup (link) {

let image = document.createElement("img");
image.src = link;

// Create a new div element to hold the image and style it as a popup
let popup = document.createElement("div");
popup.className = "popup";
popup.appendChild(image);

// Append the popup to the body of the document
document.body.appendChild(popup);

// Add an event listener to close the popup when clicked
popup.addEventListener("click", function() { document.body.removeChild(popup); });

}
  • Function should be good at one thing only, don't wear multiple hats for function.
OBJECTS

These are collections of properties and methods that represent data and behavior. Objects can be created with literals, constructors, or the Object.create() method.

Objects can inherit from other objects using prototypes or classes. Objects are the main way of organizing and encapsulating code in JavaScript.

var exampleObject = {

// This is a property that stores a string value.
name: "exampleObject",

// This is a property that stores a number value.
size: 10,

// This is a property that stores an array value.
colors: ["red", "green", "blue"],

// This is a method that returns the name of the object.
getName: function() { return this.name; },

// This is a method that changes the size of the object by a given amount.
resize: function(amount) { this.size += amount; },

// This is a method that adds a new color to the object's array of colors.
addColor: function(color) { this.colors.push(color); }

}
  • One of popular data storage format is JSON which utilize JavaScript object syntax to represent data.
